

وصف المشروع

هذا المشروع عبارة عن لعبة ثلاثية الأبعاد (3D Maze Game) تم تطويرها باستخدام ++C و OpenGL (Core Profile 3.3) بدون الاعتماد على أي محركات ألعاب جاهزة.

فكرة اللعبة تعتمد على التحكم بشخصية داخل متاهة تحتوي على جدران وعوائق، حيث يتوجب على اللاعب جمع جميع العناصر المنتشرة داخل الخريطة للوصول إلى حالة الفوز.

تم بناء المشهد باستخدام تقنيات الرسم الحديثة في OpenGL، مع استخدام Shaders لتطبيق الخامات (Textures) على الأرضيات والجدران، بالإضافة إلى عرض شاشات بداية ونهاية باستخدام صور ثنائية الأبعاد.



الية اللعب

* يبدأ اللاعب في شاشة البداية
* عند الضغط على Enter تبدأ اللعبة
* يتحرك اللاعب داخل المتاهة باستخدام لوحة المفاتيح
* يجب جمع جميع العناصر (Items) الموجودة في الخريطة
* يتم حساب الوقت (Timer) والسكور بشكل مباشر
* عند جمع كل العناصر تظهر شاشة الفوز



 التقنيات المستخدمة

* OpenGL (Modern / Core Profile)
* GLFW لإنشاء النافذة والتعامل مع الإدخال
* GLEW لإدارة دوال OpenGL
* stb_image لتحميل الصور
* Shaders (Vertex + Fragment) لتطبيق الإضاءة والخامات
* مصفوفات التحويل (Model / View / Projection)



الميزات

* نظام حالات (Start / Playing / Win)
* كاميرا ثلاثية الأبعاد مع حركة ودوران
* متاهة تحتوي على عوائق مع منع اختراق الجدران (Collision)
* عناصر قابلة للجمع (Collectibles)
* نظام نقاط (Score) وتايمر (Timer)
* تطبيق خامات (Textures) على الأرضية والجدران
* شاشات بداية ونهاية باستخدام صور
